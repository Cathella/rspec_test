
# RSpec Testing

Project to learn and get deeper into RSpec Testing.

## Setup

Step - 1

```bash
  mkdir rspec_test
```

cd into the created directory and create a Gemfile inside it.

In the Gemfile add

```bash
  source 'https://rubygems.org'
  
  gem 'rspec'
```

then continue to run in the terminal

```bash
  bundle install
```
```bash
  bundle install --binstubs
```
Step - 2

Initialize RSpec

```bash
  bin/rspec --init
```
